## FAQ for Users

**Q: Can I use DevSetup Pro at work?**
A: If your company has < 10 employees, yes for free. Otherwise, commercial license required.

**Q: Can I modify the code?**
A: Yes! You can modify for personal/internal use. Redistribution follows BSL terms.

**Q: Can I contribute?**
A: Absolutely! We welcome contributions. You'll need to sign a simple CLA.

**Q: What happens after 4 years?**
A: The code becomes Apache 2.0 licensed (fully open source).

**Q: Why not just make it fully open source?**
A: We need revenue to maintain and improve the project. BSL lets us build sustainably while staying transparent.

## Migration Plan

### If You Later Want to Fully Open Source:

**Option 1: Convert Gradually**
- Keep BSL for new features (1-2 years)
- Old versions become Apache 2.0
- Creates "freemium" model

**Option 2: Foundation Model**
- Transfer to non-profit foundation
- Accept sponsorships/donations
- Hire full-time maintainers

**Option 3: Services Model**
- Fully open source code
- Charge for hosting, support, training
- Like WordPress, Nextcloud
