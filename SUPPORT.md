# Support

## Where to Get Help

- **Bugs / errors:** Open a GitHub Issue and include logs + reproduction steps.
- **Feature requests:** Open a GitHub Issue (label: enhancement).
- **Questions:** Use GitHub Issues (or GitHub Discussions if enabled later).

## Before You Open an Issue

Please include:
- OS (Ubuntu version) or Windows + WSL (Ubuntu version)
- DevSetup Pro version
- What you selected to install/uninstall
- Copy/paste the Log Viewer output (remove any secrets)

## Security Issues

If this is a security issue, please follow [SECURITY.md](SECURITY.md).
